# Gitignore

> Git 을 통한 프로젝트 시작시 반드시 설정하자!

## 활용법

```java
*.xlss # 확장자가 xlss인 파일
  test/ # text 폴더
  a.txt # 특정파일
```

보통 .gitignore 에 등ㅇ록 되는 파일은 iDE(이클립스)설정과 관련된 내용 혹은 프로그래밍 언어별 임시 파일, 윈도우 등 OS 관련 파일을 등록한다. (프로젝트 소스코드와 무관)

잘 모르는 경우 해당 환경을 gitignore.io 에서 검색하며 설정하자.

예) eclipse, java, windows...
