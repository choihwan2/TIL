## 빅데이터?  

IEEE & ACM 논문 좋음

기존에는 3V 를 가지고있다

Volume : 데이터의 양

Velocity : 데이터 입출력 속도

Variety : 데이터 종류의 다양성

=> 프로세스의 최적화를 향상시키기 위해 새로운 형태의 처리방식 필요. => 요게 빅데이터다.

추가로 요즘에는 IBM은 => Varacity : 진실성을 추가. Brian Hopkins는 Variablility : 가변성을 추가. 4V를 만들었다. 

분석과 의견: 내 생각이 있고 없고. 즉 객관성이 있고 없고. => 결국 수학이다. 깔끔~
